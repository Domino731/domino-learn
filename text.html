When we visit a website, our web browser will format the html document according to the information in the stylesheet.

In order to have access to modify the visual layer of our page we need to know how to insert our styles into the HTML code. There are three ways of inserting styles to our .html file.

  • External CSS
  • Internal CSS
  • Inline Css

External

This is the best method. It helps us to keep our design in order. To change the look of our page we only need to change one CSS file. To add it we need to add a link in the head section of the HTML document: <link rel = "stylesheet" type = "text/css" href = "style.css" >

Internal

If you don't want to create a new stylesheet, you can add styles inside the <style> element inside the <head> section;

Inline

With inline style we can create a unique style for a single HTML element: <div style="color: red; font-size: 20px; margin-top: 10px"> <div>. This is not the recommended approach, use it as a last resort.

While styling our elements, we need to keep the hierachy in mind. For example, if a div element has a red background in a separate stylesheet and a blue background in an inline style, it will have a blue background because inline styling is the highest in the hierarchy.
