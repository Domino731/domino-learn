When we create a website, its most important thing is the content on this page. We often want to give some meaning to the text - underline, cross out, bold.Giving meaning to the text will attract the user's attention.

In the text editor, we can easily cross out text or write in italics. In HTML there are over a hundred tags to format our text. Full list can be found at html.spec.whatwg.org. However, usually only these basic tags are used. Below is a list of them.

  • <strong> - makes the text bold. It marks the fragment of text that you want to draw attention to and gives it a semantic meaning.
  • <b> - like the above, it boldens the text but doesn't give it semantic meaning.
  • <i> - makes the text that is in it become slanted. Can be used for technical words or names.
  • <small> - used to shrink text. Most commonly found in the footer of a page, e.g. for license text.
  • <del> - text surrounded by this tag becomes crossed out.
  • <span> - is a tag that does not give any meaning to the text, it is a very useful tag for styling in CSS.
